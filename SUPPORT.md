# Support

Thanks for using **Base64 Converter**! Here is how to get help.

## 📖 Documentation

- [README](README.md) — features, installation, and usage.
- [AI-generated wiki (DeepWiki)](https://deepwiki.com/DoctorLai/base64-converter).

## ❓ Questions & Help

- Browse existing [issues](https://github.com/DoctorLai/base64-converter/issues)
  to see if your question has already been answered.
- Open a new [issue](https://github.com/DoctorLai/base64-converter/issues/new/choose)
  using the most appropriate template.

## 🐞 Reporting Bugs

Please use the **Bug Report** issue template and include:

- What you expected to happen.
- What actually happened.
- Steps to reproduce.
- Your browser and operating system.

## 💡 Requesting Features

Use the **Feature Request** issue template and describe the problem you are
trying to solve, not just the solution you have in mind.

## 🔒 Security Issues

Please **do not** report security vulnerabilities in public issues. See our
[Security Policy](SECURITY.md) for private reporting instructions.

## 💬 Community

- Try the [live demo](https://doctorlai.github.io/base64-converter/).
- Like the project? Consider [buying the maintainer a coffee](https://buymeacoffee.com/y0btg5r) ☕.
